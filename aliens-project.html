Application Node - Aliens

Projet de formation DEUST T-MIC - NodeJS

Projet réalisé dans le cadre de la formation DEUST T-MIC.

Le projet imposait d'utiliser les technologies suivantes :

  • Node JS (Utilisation du framework EXPRESS)
  • HandleBars

La demande était la suivante : créer une application de rescencement d'extraterrestres.

Du point de vue administratif, il est possible de créer, supprimer et de changer les informations de chaque extraterrestre.

L'application n'utilise pas de base de données mais un fichier JSON comme sauvegarde des informations.
